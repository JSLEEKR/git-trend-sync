## 📋 What is this?

An automated system that collects trending AI agent repositories from GitHub Topics, computes quantitative metrics, performs qualitative analysis, and generates a daily report.

### Categories

| Category | Description |
|:---------|:------------|
| 🧠 **AI Agent Framework** | General-purpose AI agent frameworks (LangChain, CrewAI, AutoGen...) |
| 🔍 **RAG Framework** | Retrieval-augmented generation frameworks (LlamaIndex, RAGFlow...) |
| 🤝 **Multi-Agent** | Multi-agent orchestration systems (MetaGPT, ChatDev...) |
| 💻 **Coding Assistant** | AI coding tools (Cursor, Copilot, Aider...) |

### Analysis Includes

- **Quantitative Metrics** — Stars, activity, community health, growth, maturity (0-10 normalized)
- **Pros & Cons** — Individual strengths and weaknesses
- **Best Combinations** — Which tools work well together
- **Avoid Combining** — Which tools conflict
- **Rankings** — Category-wise rankings with justification

## 🚀 Quick Start

### Prerequisites

- Python 3.10+
- GitHub Personal Access Token

### Setup

```bash
# Clone
git clone https://github.com/JSLEEKR/ai-trend.git
cd ai-trend

# Install dependencies
pip install -r requirements.txt

# Set your GitHub token
echo "GITHUB_TOKEN=ghp_your_token_here" > .env
```

### Run

```bash
# Full pipeline (collect → metrics → analysis → report)
python run.py

# Skip analysis (metrics only)
python run.py --skip-analysis

# Regenerate report from existing data
python run.py --report-only

# No git push
python run.py --no-push
```

## ⚙️ Automation

### Windows Task Scheduler

1. Open Task Scheduler
2. Create Basic Task → Name: `AI Trend Report`
3. Trigger: Daily at your preferred time
4. Action: Start a program
   - Program: `python`
   - Arguments: `C:\path\to\ai-trend\run.py`
5. Done
