## 📋 What is this?

GitHub Topics에서 AI 에이전트 관련 상위 레포지토리를 수집하고, 정량/정성 분석을 수행하여 매일 리포트를 생성하는 자동화 시스템입니다.

### Categories

| Category | Description |
|:---------|:------------|
| 🧠 **AI Agent Framework** | 범용 AI 에이전트 프레임워크 (LangChain, CrewAI, AutoGen...) |
| 🔍 **RAG Framework** | 검색 증강 생성 프레임워크 (LlamaIndex, RAGFlow...) |
| 🤝 **Multi-Agent** | 멀티 에이전트 오케스트레이션 (MetaGPT, ChatDev...) |
| 💻 **Coding Assistant** | AI 코딩 도구 (Cursor, Copilot, Aider...) |

### Analysis Includes

- **Quantitative Metrics** — Stars, activity, community health, growth, maturity (0-10 normalized)
- **Pros & Cons** — Individual strengths and weaknesses
- **Best Combinations** — Which tools work well together
- **Avoid Combining** — Which tools conflict
- **Rankings** — Category-wise rankings with justification

## 🚀 Quick Start

### Prerequisites

- Python 3.10+
- [Claude Code CLI](https://claude.ai/download)
- GitHub Personal Access Token

### Setup

```bash
# Clone
git clone https://github.com/JSLEEKR/ai-trend.git
cd ai-trend

# Install dependencies
pip install -r requirements.txt

# Set your GitHub token
echo "GITHUB_TOKEN=ghp_your_token_here" > .env
```

### Run

```bash
# Full pipeline (collect → metrics → Claude analysis → report)
python run.py

# Skip Claude analysis (metrics only)
python run.py --skip-analysis

# Regenerate report from existing data
python run.py --report-only

# No git push
python run.py --no-push
```

## ⚙️ Automation

### Windows Task Scheduler

1. Open Task Scheduler
2. Create Basic Task → Name: `AI Trend Report`
3. Trigger: Daily at your preferred time
4. Action: Start a program
   - Program: `python`
   - Arguments: `C:\path\to\ai-trend\run.py`
5. Done
